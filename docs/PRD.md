# Product Requirements Document (PRD)

## Project Name

Smart Meeting Agent – Recruitment Automation System

## Purpose

Develop an automated recruitment system that simulates candidate applications, processes incoming Gmail messages, evaluates candidates, and generates recruitment reports.

## Business Problem

Recruitment teams spend significant time reviewing candidate applications manually. The system automates candidate intake and initial evaluation.

## Objectives

* Generate candidate applications automatically.
* Send applications through Gmail API.
* Read candidate emails automatically.
* Extract candidate information.
* Score and rank candidates.
* Produce Excel-based recruitment reports.

## Inputs

* Candidate database (Excel).
* Gmail inbox messages.

## Outputs

* Candidate ranking report.
* Excel summary files.
* Recruitment statistics.

## Technologies

* Python
* Gmail API
* OAuth 2.0
* Pandas
* OpenPyXL

## Success Criteria

* Successful Gmail authentication.
* Automated email processing.
* Candidate ranking generation.
* Excel report creation.
